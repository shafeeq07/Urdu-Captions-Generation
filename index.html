Urdu Captions Generation

Introduction

In recent years, Artificial Intelligence plays importance role in solving real word problems. Automatic image caption generation is one of the areas of Artificial Intelligence which attracted many researcher attentions. The task is very challenging, since it requires combination of Compute Vision and Natural Language Translation techniques to automatically generate sentence description for an image. Automatic image captioning plays important role in computer vision applications, i.e., helping the visually impaired people to visualize the world in the future, enabling computer to understand the images, human computer interaction, video tracking. Urdu is Pakistan national language and also spoken in nearby countries like India, Bangladesh, Nepal and few work has been done for Urdu caption generation which have huge room for improvement. In recent work, researchers achieved great success in image captioning for many common languages like English. This process requires large-scale corpora to describe image into any language. The corpora is publicly available for some common languages like English. Unfortunately, It is not available for many languages like Urdu. Our research aims to developing a technique to achieve human level accuracy for Urdu caption generation which leads to help people who only understand Urdu.

Dataset

Flicker8k dataset is avalaible publicaly on the internet containing 8000 images and 5 captions for each image. We translated a small portion of Flicker8k dataset from English to Urdu. Our Urdu Captions dataset contains almost 6595 captions for 1319 images, 5 captions per Image. We are continuously translating Captions to increase the size of dataset.
